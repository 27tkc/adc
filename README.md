## 🦷 Dental Clinic Web App

ADC stands Amit Dental Clinic

A modern, responsive dental clinic application built using **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.
This project focuses on building a **scalable frontend architecture** with **reusable UI components** and clean design patterns used in real-world applications.

---

## ✨ Features

- 🧱 Reusable UI Component Library (Button, Input, Card, Modal, Navbar)
- 📄 Multi-page layout with Next.js App Router
- 📱 Fully responsive design
- 🧾 Appointment booking UI (modal-based)
- ⚡ Clean and scalable folder structure
- 🎯 Component-driven development approach
- 🎨 Styled with Tailwind CSS

## 🛠️ Tech Stack

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **ESLint**

## 🎯 Purpose of the Project

This project is built as part of frontend practice to:

- Strengthen **React and Next.js fundamentals**
- Learn **component reusability and design systems**
- Simulate a **real-world production-ready application**
- Build a strong **portfolio project for frontend roles**
